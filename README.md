## Branchを切る

$ git checkout -b feature-readme

## add, commit, pushする

$ git add . 
$ git commit -m "add README.md" 
$ git push origin feature-readme     

## pull requestsをgithub上で作成し、ペアの人に見てもらう

OKだったら、マージし、github上のBranchを消去する。
ローカルでも、使用が終わったBranchに関しては消去する。
